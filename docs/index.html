Fundamentação

Arquitetura do Problema

Orquestração Dinâmica

O sistema em Python simula o comportamento de sensores espaciais, controla o tempo de execução e cria modelos em 3D para mostrar onde a estrutura sofreu deformações.

Dividir e Conquistar

Ordenação por coordenadas e varredura recursiva de par mais próximo reduzem a complexidade de O(n²) para O(n log n).

Processamento

Pipeline do Sistema

Etapas lógicas utilizadas pelo backend para mapear e analisar áreas de tensão.

01

Ingestão Cartesiana

O orquestrador gera as coordenadas 3D de telemetria dos sensores e injeta via stream otimizado diretamente na memória compartilhada do motor nativo.

02

Ordenação por Eixo

O algoritmo realiza um pré-processamento ordenando todos os pontos com base na coordenada X para permitir a divisão espacial exata pelo plano mediano.

03

Divisão Recursiva (Sub-problemas)

O espaço é dividido ao meio. Metades independentes são delegadas para processamento recursivo sequencial otimizado que calcula a distância mínima interna de cada lado.

04

Varredura de Fronteira (Strip)

O algoritmo cria uma faixa vertical de largura 2δ ao redor da linha de divisão. Os pontos contidos nessa faixa são ordenados pelo eixo Y, comparando apenas os vizinhos necessários.

05

Retorno

Consolida o menor valor absoluto encontrado entre as metades e a faixa de fronteira, retornando os IDs dos dois sensores críticos para o plot geométrico final.

#include "point.h"
#include <algorithm>
#include <cmath>

double distancia(const Point& p1, const Point& p2) {
    return std::sqrt((p1.x - p2.x)*2 + (p1.y - p2.y)*2 + (p1.z - p2.z)*2);
}

std::pair<Point, Point> parMaisProximoRecursivo(std::vector<Point>& Px, std::vector<Point>& Py) {
    if (Px.size() <= 3) return forcaBruta(Px);

    int meio = Px.size() / 2;
    Point pontoMedio = Px[meio];

    std::vector<Point> PxEsq(Px.begin(), Px.begin() + meio);
    std::vector<Point> PxDir(Px.begin() + meio, Px.end());

    // Divisão e Conquista Recursiva Padrão
    auto par_esq = parMaisProximoRecursivo(PxEsq, Py);
    auto par_dir = parMaisProximoRecursivo(PxDir, Py);

    auto resultado = (distancia(par_esq.first, par_esq.second) < distancia(par_dir.first, par_dir.second)) ? par_esq : par_dir;
    return checarFaixaFronteira(Py, pontoMedio, resultado);
}

src/orquestrador.py Python 
import subprocess
import random

def executar_benchmark_iot(num_sensores):
    # Geração estocástica de coordenadas espaciais 3D
    coordenadas = []
    for i in range(num_sensores):
        x = random.uniform(0.0, 1000.0)
        y = random.uniform(0.0, 1000.0)
        z = random.uniform(0.0, 500.0)
        coordenadas.append(f"{i} {x} {y} {z}")
        
    # Chamada IPC via subprocess enviando os dados para a Engine C++
    processo = subprocess.Popen(['./motor'], stdin=subprocess.PIPE, stdout=subprocess.PIPE, text=True)
    saida, _ = processo.communicate(input="\n".join(coordenadas))
    
    return saida
